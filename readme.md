# EcoMap – ODS 11
Plataforma colaborativa para mapear pontos de reciclagem, áreas verdes e problemas ambientais em Foz do Iguaçu/PR.

✅ Alinhado com ODS 11 (Cidades e Comunidades Sustentáveis)  
✅ Projeto de Extensão – Análise e Desenvolvimento de Sistemas  
✅ Tecnologias: HTML, CSS, JavaScript, Leaflet.js  
✅ Deploy gratuito no GitHub Pages

## Como usar
1. Clone ou baixe o repositório
2. Abra `index.html` no navegador
3. Adicione pontos clicando no formulário (coordenadas simuladas)
4. Veja o mapa interativo com marcadores
